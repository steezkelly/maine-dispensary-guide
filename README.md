## 🧠 Core Intelligence Engines
### 1. Link Architect 2.0 (`scripts/link-architect.cjs`)
A high-fidelity Node.js engine that automatically scans guide content and injects glossary links.
- **Body-Only Injection:** Protects metadata and layout props.
- **Regex Logic:** Ensures terms like "Metrc" and "280E" are linked only once per page to prevent SEO over-optimization.

### 2. ROI Success Calculator (`src/pages/roi-calculator.astro`)
An interactive "Business Flight Simulator" that uses real Maine market data:
- **COGS:** 65% (Retail Standard).
- **280E Tax:** 21% effective rate on Gross Profit.
- **OPEX:** Monthly burn rate tracking.

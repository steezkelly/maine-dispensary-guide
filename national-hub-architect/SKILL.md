# National Hub Architect

This skill provides the strategic and technical framework for building "Information Gain" digital empires.

## Core Directives

### 1. Information Gain (SEO 2026)
Prioritize unique, verifiable data over raw word count. 
- **Knowledge Base:** Refer to [references/GEO_STANDARDS.md](references/GEO_STANDARDS.md) for detailed rules.
- **Action:** Add one unique data point (Local X-Factor) that top-ranking competitors lack.

### 2. GEO Engineering (AI-Discovery)
Optimize for Generative Engine citations (ChatGPT, Perplexity):
- **Fact Boxes:** Lead every technical guide with the [assets/fact-box-template.astro](assets/fact-box-template.astro).
- **First 200 Words Rule:** The primary intent answer must be in the first 200 words.

### 3. Technical Integrity (Node-First)
- Use Node.js (.cjs) for site-wide string manipulation.
- Avoid brittle shell scripts for complex architectural automation.

## Workflows

### Phase 1: High-Density Content
1. Scan for "Underweight" content (less than 800 dense words).
2. Pivot to **Factual Density** using the template asset.
3. Inject specific legal citations (e.g., M.R.S. Title 28-B).

### Phase 2: Knowledge Web Sync
Run the bundled Link Architect engine to link technical terms without breaking metadata.

### Phase 3: Multi-Agent Continuity
Maintain a `BOT_COLLABORATION_HUB.md` with EDT timestamps to ensure context is never lost.

## Operational Scripts
- **Link Architect:** `node national-hub-architect/scripts/link-architect.cjs`
- **Health Check:** `powershell.exe -File national-hub-architect/scripts/health-check.ps1`
